自驱闭环 Loop / 产物流转与返工主控图

            

              
01人工输入
原著/创意/风格/成片标准进入总导演调度。
director.message

              
02任务规划
总导演生成 TaskGraph，拆分剧本、美术、分镜、视频、音剪线。
taskgraph.created

              
03Agent 执行
各生产 Agent 产出剧本、设定、提示词、图像、视频、音频。
task.started / task.completed

              
04业务回写
CallbackBridge 写回 Toonflow 业务表并生成版本元数据。
callback.persisted / version.created

              
05专项审核
剧本/角色/场景/提示词/图片/视频/音画同步逐项打分。
review.scored

              
06打回重跑
未通过则生成 retryInstruction，原路或跨工种打回。
review.reroute

              
07总监制复核
阶段完成后汇总报告，判断是否进入人工终审。
director.user_input_required

              
08人工终审
用户只在关键门禁决定通过、打回、暂停或人工修改后再审。
director.user_input

              
09归档推进
通过后解锁下一生产线，所有版本进入追溯链。
stage.approved

            

            

              
单节点自循环
每个生产 Agent 都必须被专属审核 Agent 校验，不通过自动回到原生产工位。
分镜生图执行 Agent -> 图片质量审核 Agent -> 未通过 -> retryInstruction -> 分镜生图执行 Agent v04

              
跨工种打回
如果问题根因来自上游设定，质量监制可跨工种打回，而不是只重跑当前节点。
图片质量审核发现空间纵深不足 -> review.reroute -> 场景概念 Agent 补充 SCN-04 光影/纵深 -> 分镜提示词优化 Agent 重写约束

              
环节终审门禁
剧本、美术、分镜、视频、成片阶段完成后，由总监制汇总并请求人工终审。
专项审核全通过 -> 总监制 Agent 汇总 -> director.user_input_required -> 用户通过 -> 解锁下一生产线
